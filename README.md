# sit_volleyball_video

部員向け 試合動画管理アプリ

## 概要
部では動画共有をYouTube上で行います！

他チームに動画が漏れないよう限定公開でアップするので、URL経由でしか動画にアクセスできません。

そのためURLを管理するアプリを作りました！
